# SEC Insider Trading Tracker 📈

## Overview
A data analysis project tracking CEO and CFO insider trading patterns 
across 20 major S&P 500 companies over 11 years (2015-2026).

Built using real SEC EDGAR Form 4 filings to answer one question:
**Do stock prices go up after executives buy their own shares?**

## Tools & Technologies
- Python 3 — data collection and analysis
- SEC EDGAR API — real regulatory filing data, completely free
- Pandas — data cleaning and analysis
- Alpha Vantage API — historical stock price data
- Tableau Public — interactive dashboard

## Companies Tracked
20 companies across 8 sectors including Apple, Microsoft, Tesla, 
JPMorgan Chase, Nike, Nvidia, Goldman Sachs and more.

## Project Phases
- ✅ Phase 1 — Project setup and environment
- 🔄 Phase 2 — SEC EDGAR data collection (in progress)
- ⏳ Phase 3 — Stock price data collection
- ⏳ Phase 4 — Data cleaning and analysis
- ⏳ Phase 5 — Tableau dashboard
